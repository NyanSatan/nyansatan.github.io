What's dualbootstuff?

Package of tools for iOS dualbooting
  • hfs_resize resizes HFS+ volumes
  • gptfdisk is interactive tool for editing GPT
  • asr, asr5, asr4 are Apple System Restore. Restores DMG images to block devices. There're versions from iOS 6.1.3, iOS 5.0.1, iOS 4.0
  • kloader loads arbitrary iBoot images using iOS deep sleep
  • kloader_ios5 is a fork of kloader with iOS 4/5 support and automatic awakening
  • multi_kloader loads two images instead of one
  • 4smulti_kloader is fixed for A5 devices
  • ibsspatch patches iOS 4 iBoot
  • attach, detach attaches/detaches a DMG image
  • ttbthingy patches kernel to re-enable use of UID-key. Required to downgrade keybags
  • fixkeybag generates/downgrades/decrypts system keybags
  • NoMoreSIGABRT patches attributes of a HFS+ volume to add the "protect" flag
  • TwistedMind2 repartitions iOS style GPT
