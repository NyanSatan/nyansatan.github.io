What's Way Out?

Way Out is a GUI for kloader/multi_kloader in the spirit of iOS 5/6 setup assistant and designed for dualboot to these versions (you can boot arbitrary iOS with this tool of course)

Features

  • multi_kloader (on iOS 6+)
  • kloader
  • Preboot scripting
