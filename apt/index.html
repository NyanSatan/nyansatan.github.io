Packages
Way Out
Way Out is a GUI for kloader/multi_kloader in the spirit of iOS 5/6 setup assistant and designed for dualboot to these versions (you can boot arbitrary iOS with this tool of course)

Features

  • multi_kloader (on iOS 6+)
  • kloader
  • Preboot scripting

dualbootstuff
Package of tools for iOS dualbooting, including:
  • hfs_resize resizes HFS+ volumes
  • gptfdisk is interactive tool for editing GPT
  • asr is Apple System Restore. Restores DMG images to block devices
  • kloader loads arbitrary iBoot images using iOS deep sleep
  • multi_kloader loads two images instead of one
  • attach, detach attaches/detaches a DMG image
  • ttbthingy patches kernel to re-enable use of UID-key. Required to downgrade keybags
  • fixkeybag generates/downgrades/decrypts system keybags
  • NoMoreSIGABRT patches attributes of a HFS+ volume to add the "protect" flag
  • TwistedMind2 repartitions iOS style GPT (for iOS 4.x dualboot)
